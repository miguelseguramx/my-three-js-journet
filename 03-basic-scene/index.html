Basic Scene with Three.js

Elements to get started with three.js

  • A scene that will contain the objects
  • Some objects
  • A camera
  • A renderer

Scene

  • Like a container
  • We put objects, models, lights, etc. in it
  • At some point we as three.js to render that scene

Objects

Can be many things

  • Primitive geometries
  • Imported models
  • PArticles
  • Lights
  • Etc.

Camera

  • Not visible
  • Serve as a point of view when doing a render
  • Can have multiple and switch between them
  • Different types
